## [2.1.0] - 2026-03-09

### ✨ 新增 (Features)

- **🤖 自动跟单系统** - 基于引力指数智能筛选并自动跟随优质交易者
  - 三种预设策略：保守型、平衡型、激进型
  - 实时复制目标交易者开平仓操作
  - 自动仓位调整和风险管理
  
- **🛡️ 多层风险控制** - 保护本金的完整风控体系
  - 交易者筛选层：引力指数、胜率、回撤门槛
  - 资金管理层：单仓上限、总仓位限制
  - 实时监控层：日亏损限制、最大回撤监控
  - 紧急保护层：自动止损、黑天鹅保护
  
- **📈 收益追踪系统** - 实时统计跟单表现
  - 总收益率、日收益率、最大回撤
  - 夏普比率、胜率、盈亏比
  - 分交易者收益统计
  - 周度/月度报告生成
  
- **⚡ 智能调仓** - 根据市场情况动态调整
  - 自动识别高信用分交易者
  - 动态调整跟随仓位大小
  - 策略切换支持

### 🔧 改进 (Improvements)

- 更精准的信用评分算法
- 优化的 API 调用流程
- 改进的错误处理和日志
- 更详细的示例文档

### 📚 文档 (Documentation)

- 新增 [COPYTRADING.md](COPYTRADING.md) 完整跟单文档
- 更新 README.md 添加跟单功能介绍
